# Fort Resource

Modders resource. Two imperial fort interiors that were part of an abandoned project, now available for use in your work.

## Interior Cells

+ AAFRFort01
  - Navmesh: True
+ AAFRFort02
  - Navmesh: True
  - Variation Markers: AAFR02ImperialMarker, AAFR02StormcloakMarker
